Use this page to generate and display a QR code for MFA setup

Note: This is a static reference page. The actual MFA QR code is generated dynamically on the /mfa/enroll page during the authentication flow.

Getting Started

  1. Sign up at http://localhost:8000/signup
  2. Login at http://localhost:8000/login
  3. Follow the MFA setup flow
  4. Scan the QR code with your authenticator app
  5. Enter the 6-digit code

Manual Entry

If you can't scan the QR code, you can manually enter the secret key shown during the enrollment process.

Secret will be provided during enrollment

Supported Apps:
  • Google Authenticator
  • Authy
  • Microsoft Authenticator
  • 1Password
  • Bitwarden
